# Outlook Tag Processor

Add-in Outlook pour convertir les tags `[xxx]` en catégories.

## Installation

1. Modifiez `manifest.xml` : remplacez `VOTRE_USERNAME` par votre nom d'utilisateur GitHub
2. Activez GitHub Pages dans les paramètres du repository
3. Déployez l'add-in via le Centre d'administration Microsoft 365

## Utilisation

1. Sélectionnez un mail avec des tags dans le sujet (ex: `[Projet-A] [Urgent] Réunion`)
2. Cliquez sur le bouton "Traiter les tags"
3. Les catégories sont créées et appliquées automatiquement
